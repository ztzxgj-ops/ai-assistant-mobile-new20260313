## 🔧 技术细节

### 修复原理

**问题代码**:
```dart
// 这个模式不处理空字符串
final title = widget.plan['title']?.toString() ?? widget.plan['content']?.toString() ?? '';
```

**修复代码**:
```dart
// 这个模式处理空字符串和null
final titleStr = widget.plan['title']?.toString() ?? '';
final contentStr = widget.plan['content']?.toString() ?? '';
final title = titleStr.trim().isEmpty ? contentStr : titleStr;
```

### Dart 字符串处理的关键区别

```dart
// 情况 1: title 为 null
String? title = null;
String result = title?.toString() ?? 'default';  // 结果: 'default' ✅

// 情况 2: title 为空字符串 ""
String? title = '';
String result = title?.toString() ?? 'default';  // 结果: '' ❌ (不是 'default')

// 修复方案
String? title = '';
String titleStr = title?.toString() ?? '';
String result = titleStr.trim().isEmpty ? 'default' : titleStr;  // 结果: 'default' ✅
```
