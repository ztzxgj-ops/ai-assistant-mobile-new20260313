# 阿里云邮件推送配置 - 快速操作清单

## 🎯 你的配置信息

根据你的情况，建议使用以下配置：

- **邮件域名**: mail.assistant230.top（或 mail.你的域名.com）
- **发信地址**: noreply@mail.assistant230.top
- **发件人名称**: AI个人助理

---

## ✅ 操作清单（按顺序完成）

### 1️⃣ 开通邮件推送服务
- [ ] 访问：https://dm.console.aliyun.com
- [ ] 点击"立即开通"（如果未开通）
- [ ] 选择"按量付费"
- [ ] 确认开通成功

### 2️⃣ 添加发信域名
- [ ] 在邮件推送控制台，点击"发信域名"
- [ ] 点击"新建域名"
- [ ] 输入域名：mail.assistant230.top（或你的域名）
- [ ] 点击"确定"

### 3️⃣ 配置DNS记录
- [ ] 访问：https://dns.console.aliyun.com
- [ ] 找到你的域名，点击"解析设置"
- [ ] 添加TXT记录（SPF）：
  ```
  主机记录：mail
  记录值：v=spf1 include:spf.mxhichina.com -all
  ```
- [ ] 添加MX记录：
  ```
  主机记录：mail
  记录值：mxn.mxhichina.com
  优先级：10
  ```
- [ ] 添加CNAME记录（从邮件推送控制台复制）

### 4️⃣ 验证域名
- [ ] 等待10-30分钟让DNS生效
- [ ] 运行检查命令：`./check_dns.sh`
- [ ] 在邮件推送控制台点击"验证"
- [ ] 确认状态变为"验证通过"

### 5️⃣ 创建发信地址
- [ ] 在邮件推送控制台，点击"发信地址"
- [ ] 点击"新建发信地址"
- [ ] 填写：
  - 发信地址：noreply@mail.assistant230.top
  - 发件人名称：AI个人助理
- [ ] 点击"确定"

### 6️⃣ 创建AccessKey
- [ ] 访问：https://ram.console.aliyun.com
- [ ] 点击"用户" → "创建用户"
- [ ] 填写：
  - 登录名称：dm-api-user
  - 访问方式：✓ OpenAPI调用访问
- [ ] 保存AccessKey ID和Secret（⚠️ 只显示一次！）
- [ ] 添加权限：AliyunDirectMailFullAccess

### 7️⃣ 更新配置文件
- [ ] 运行：`python3 setup_email_wizard.py`
- [ ] 输入AccessKey ID
- [ ] 输入AccessKey Secret
- [ ] 输入发信地址
- [ ] 确认保存

### 8️⃣ 安装依赖
- [ ] 运行：`pip3 install aliyun-python-sdk-core aliyun-python-sdk-dm`

### 9️⃣ 测试配置
- [ ] 运行：`python3 check_email_config.py`
- [ ] 确认显示"生产模式"
- [ ] 运行：`python3 test_email_send.py`
- [ ] 输入测试邮箱
- [ ] 检查是否收到邮件

### 🔟 重启服务
- [ ] 本地：`python3 assistant_web.py`
- [ ] 或生产服务器：`sudo supervisorctl restart ai-assistant`
