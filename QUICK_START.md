## ✨ 首次使用

1. **启动应用** → 使用上面任一方式
2. **登录/注册** → 输入用户名和密码
3. **进入设置** → 点击右上角设置按钮
4. **选择提醒模式** → 选择 **"本App提醒"**
5. **创建提醒** → 输入 "每天18:00提醒测试"
6. **验证** → 应该看到提醒在列表中，不再显示错误

## 🔧 常见问题

| 问题 | 解决方案 |
|------|--------|
| 应用无法打开 | 系统设置 → 隐私与安全 → 通用 → 允许打开 |

| 提醒创建失败 | 重启应用，检查提醒模式设置 |
| 权限被拒绝 | 系统设置 → 隐私与安全 → 检查权限 |
