## 🚀 使用步骤

### 1. 安装依赖（npm正在后台运行，请等待完成）
npm install

### 2. 运行应用
npm start

### 3. 打包Mac应用
npm run build:mac

输出文件：dist/AI个人助理.dmg

## ✨ 功能特点

- 独立桌面应用，不依赖浏览器
- 自动连接到云服务器
- 系统托盘支持
- Mac原生窗口体验
- 完全复用Web版所有功能

## 📝 注意事项

1. 需要网络连接才能使用
2. 首次打包需要下载Electron二进制文件（约100MB）
3. 打包的应用大小约150MB（包含Chromium）

## 🔧 配置

如需修改服务器地址，编辑 main.js 第10行：
const CLOUD_SERVER_URL = 'http://你的服务器地址/ai/';
