## 🐛 修复的问题

### 问题：UTF-8编码错误

**错误信息**：
```
'utf-8' codec can't decode bytes in position 661-662: invalid continuation byte
```

**原因**：
Shell脚本输出包含ANSI颜色代码或其他特殊字符，Python默认的UTF-8解码失败。

**解决方案**：
```python
# 修改前
subprocess.Popen(
    ['bash', script_path],
    stdout=subprocess.PIPE,
    stderr=subprocess.STDOUT,
    universal_newlines=True  # ❌ 可能出错
)

# 修改后
subprocess.Popen(
    ['bash', script_path],
    stdout=subprocess.PIPE,
    stderr=subprocess.STDOUT,
    encoding='utf-8',
    errors='replace'  # ✅ 遇到无法解码的字节时替换为�
)
```

**效果**：即使Shell脚本输出包含特殊字符，也不会导致备份失败。
