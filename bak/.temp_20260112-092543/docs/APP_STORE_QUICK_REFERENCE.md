## 🛠️ 快速命令

### 清理和准备
```bash
cd ai-assistant-mobile
flutter clean
rm -rf ios/Pods ios/Podfile.lock
flutter pub get
cd ios && pod install --repo-update && cd ..
```

### 构建Release
```bash
flutter build ios --release
```

### 打开Xcode
```bash
open ios/Runner.xcworkspace
```

### 验证Bundle ID
```bash
grep -r "com.gaojun.aiassistant" ios/
```

## ⚠️ 常见错误及解决

| 错误 | 解决方案 |
|-----|--------|
| Code Signing Error | 检查Team选择和证书安装 |
| Pod Install失败 | `pod repo update` 后重试 |
| Bundle ID不匹配 | grep -r确保所有地方一致 |
| 后端连接失败 | 检查Info.plist中的NSAppTransportSecurity |
| 隐私政策URL无效 | 确保URL可以在浏览器中打开 |
